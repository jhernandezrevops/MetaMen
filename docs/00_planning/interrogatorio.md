```
Acción 1: Completar el 100% de tareas de un día: +1 Corazón.

**P6.3.1** ¿Qué pasa cuando la vida llega a CERO?
```
Ocurre el "Colapso de Identidad". El avatar muere visualmente y el sistema se bloquea por 24 horas como "Día de Duelo" y reflexión.
```
**P6.3.2** ¿Es permanente o hay forma de "resucitar"?
```
Hay resurrección mediante un "Reset de Identidad". El usuario debe reiniciar su camino, pero no desde el vacío absoluto, sino como un aprendizaje.
```
**P6.3.3** Si hay reset, ¿qué se MANTIENE?
```
- Se mantiene el historial de XP total (Proof of Work).
- Se mantienen los activos físicos comprados (Propiedades, Autos, Relojes), pero quedan "Bloqueados por Nivel" hasta que el usuario recupere el nivel necesario para equiparlos.
```
**P6.3.4** Si hay reset, ¿qué se PIERDE?
```
 El Nivel de Usuario vuelve a Nivel 1 (Indigente).
- Todos los vectores visuales (Aura, Jawline, Physique) vuelven a nivel 1.
- Se pierde el 50% de los Bitcoins acumulados (como penalización por quiebra).
- La racha actual se reduce a 0.
```
**P6.3.5** ¿Cuántas veces puede morir un usuario?
```
Infinitas, pero cada muerte consecutiva aumenta el costo de Bitcoins para la resurrección y genera una "Cicatriz Digital" visible en el perfil del avatar.
