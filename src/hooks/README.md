# Hooks

Custom React hooks de METAMEN100.

## Hooks Disponibles

- `useAvatar` - Estado y mutaciones del avatar
- `useTasks` - Gestión de tareas
- `useAuth` - Autenticación con Supabase
- `useTheme` - Tema de la aplicación

## Convenciones

- Prefijo `use` obligatorio
- Un hook por archivo
- Documentar con JSDoc
